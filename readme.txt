Let the world know what music and iPhone/iPad/iPod apps you like using a widget + editor plugin.

== Description ==

Easily insert links to iTunes products within your blog posts and show your favorites in a sidebar widget.  Formatting configuration allows you to choose if the icon for a product is displayed, if you want to use an affiliate ID, etc.

1. Use the simple button in the editor to import a song/app/album/etc. from iTunes.
1. The product will be added to your favorites.  Favorites automatically appear in the widget.  You can also manage your favorites and press a single button to insert a link to one of them from within your blog post.
1. Change display options for the widget, or add an affiliate ID using the settings.

== Installation ==

= Installation =
1. Download the zip and install it from the 'Add New' plugins section in the admin area --OR--
1. Download the zip file and extract the contents.
1. Upload the 'itunesfavorites' folder to your plugins directory (wp-content/plugins/).
1. Use the 'Plugins' page in the WP Admin area to activate the plugin
1. See 'Settings->iTunes Favorites' to specify an affiliate ID or change post display settings
1. Use 'Appearance->Widgets' to change the display options on the widget (number of items shown, etc.)

= Using iTunes Favorites to embed videos =
1. Open the iTunes application and search for the item you are looking for
1. Right-click (Mac: cmd+click) on the item in iTunes and select 'Copy Link'
1. Start a new blog post (or edit an existing one) and press the iTunes icon
1. Select the "Import New App/Music" tab and paste the link you copied

== Frequently Asked Questions ==

= Why use this plugin?
Not only can you insert cool items directly into your blog with clean tags, but you can also share your favorite app/music recommendations with your readers via a sidebar widget.
= What is an iTunes Affiliate?
iTunes affiliates earn 5% commission whenever they send a customer to the store who then buys the product which was linked to.
= How do I become an iTunes Affiliate?
http://www.apple.com/itunes/affiliates/
= How do I setup the plugin to use my affiliate account?
Please see the settings section of the plugin.
